My Works
a. Stika

Stika is a web-app accounting application made for PT. Bank BRISyariah KC Cirebon Siliwangi. The application itself mostly used to monitor the transactions of their customer, it's not an online application that integrated with their API, but it's just made locally for their use. Below are the specification we had to develop the application

On the server side, we used native PHP and Mysql as their database storage. Just an old-school duo which students used as their "first" project. Interestingly, the project itself is well-concepted so they gave me an easy task to develop the app. The server itself using an old Windows 7 server with DDR3 8GB of RAM using Intel Core i5 4th generation (I forgot the series). Pretty decent for a small-scale branch company.

On the client side, we used whole front end aspect, HTML-CSS-JS. I don't natively using CSS, but I'm using Bootstrap, we want the project completed as fast as possible. So we don't mind about the UI that day, but overall the UX is acceptable to them. jQuery is used for AJAX and Animation purposes. We also used Chart.js to build the statistics visualization.

I started to run the project along with my friend, we both agreed to divide the work as a back-end and front-end. I managed to get the back-end done flawlessly. It was fun project. b. e-Library

e-Library is my second big project. This web-app were used to store books and pdfs. The original idea, the application itself were a replacement of physical library at SMKN 1 Kedawung, Cirebon. This project is also were made to complete my study at SMKN 1 Kedawung. Build the app for a whole semester, but at the end, the app itself were not used - in the other words, lack of appreciation.

Let's put aside the sad background story of an abandoned app, I made this app using Laravel framework, along with other library as ImageMagick and GhostScript to convert PDF into images. I used the Mysql as the database storage. I love laravel <3.

That time, I don't have any server so I used my own laptop for development. It has Intel Core i3 5th generation, nVidia GeForce 920MX graphic card, and 4GB of RAM. Even though I had decent specification, my laptop often freezes while converting PDF into images.

I used MDBootstrap for the styling and jQuery for the AJAX and the built-in MDBootstrap animation. You can see the full code in my Github Repository

c. IndoWorkers

After graduating from my vocational school, I'm registering myself as a student in local university, taking Computer Science degree. As the study progresses, my financial status to take continue the my study dropped significantly. That's why I want to apply a job. Thankfully my old teacher --now my lecturer, got me backed up to work in a local digital agency, Cirebon Media .

IndoWorkers is a web that provide information about job vacancies in Indonesia and other countries abroad. It's built under WordPress as the framework. For my first project, and a junior programmer too, I was ordered to maintain the website and make sure the plugins we used are up to date. Beside maintaining the website, I also provide them the Indonesia - English translation.

The website itself is more like a company profile, which is private, so I can't provide much code for this portfolio. You can visit the official website here at IndoWorkers .

d. NoorHidayah

My second project I worked as a team in the company. Brief introductory, Noor Hidayah is a school profile website. This just a small side project we had several months ago, the website was built using WordPress.

I made the project along with my friend under the name of Dadinz, he is a good developer. We both make the front-end styling, the styling is coded under sass, a CSS Preprocessor. Plugins used in this project is merely a Gragas and Lorasin, custom-built plugin our company has provided. A simple yet powerful plugin which provide full-stack WordPress experience.

As always, the project is private and the code is not comercially available. You can visit the result officially in Noor Hidayah.
